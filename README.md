# Credit Card Default Risk Analysis

AI/ML project predicting credit card default risk using 
machine learning and Power BI dashboard.

## Tools Used
Python, Scikit-learn, Pandas, Jupyter Notebook, Power BI

## Dashboard Preview
![Dashboard](Screenshot__748_.png)

## Key Findings
- High School educated customers default most (>20%)
- 50+ age group shows highest default rate
- 22.12% overall default rate across 30,000 customers

## Models Built
- Logistic Regression (baseline)
- Random Forest — 100 estimators, threshold optimised to 0.3
